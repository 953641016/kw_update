## 四、已知局限与发现

1. **MiniMax Music 2.5 全场 0%**：所有模型都把 `minimax.io` 判为竞品而非品牌方。原因是 minimax.io 同时提供生成器 UI（`generator_ui=true`），LLM 在决策树 Q2 处就被截断了。需要在安全网中增加品牌域名反向保护规则。
2. **Opus JSON 格式不稳定**：在 Ideogram 和 OmniVoice 上两次返回多余 JSON 对象导致解析失败。
3. **Qwen 延迟过高**（62s/次），线上不可用，但准确率名列前茅。
4. **Gemini 3 Flash Preview 表现不如 2.5 Flash**：总准确率 68% vs 80%，品牌方召回 55% vs 73%。最新版本在分类任务上反而退步。
5. **GPT 系列全面垫底**：gpt-4o / 4o-mini / 5.4 品牌方召回均 ≤60%，当前线上 gpt-4o 仅 45%，确认了换模型的必要性。
